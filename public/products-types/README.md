# Images par type de produit (maison)

Ajoutez ici les photos pour les types de produits (Lave-linge, Lave-vaisselle, etc.).

**Convention de nommage :** `{slug}.jpg` ou `{slug}.png`

Exemples :
- `lave-linge.jpg`
- `lave-vaisselle.jpg`
- `refrigerateur.jpg`
- `chauffe-eau.jpg`
- `tondeuse.jpg`

Le slug = nom du type en minuscules, sans accents, avec des tirets.
Ex : "Lave-linge" → `lave-linge.jpg`

Vous pouvez aussi définir des URLs personnalisées dans `src/data/product-type-images.js`.
